# 暨南大学互联网新技术课程设计
项目背景是针对给定的大数据（中小学生问卷调查数据），选一个感兴趣的方向做设计。

我们的项目方向是数据分析与数据预测。

## 使用说明

请确保环境已安装：

- Python3
- Python Virtualenv库

```bash
cd ${PROJECT_DIR}
virtualenv venv						# 初始化虚拟环境
source activate						# 切换虚拟环境
pip3 install -r requirements.txt	# 安装必要库
jupyter notebook					# 进去后运行src.ipynb即可
```
